Installation Instructions

  1. Subcribe to the collection and Download the mods External mods linked above
  2. Locate your mod folder
  3. Copy the downloaded file to the appropriate location
  4. Launch the game and enable the mods

Installation Location

  1. \Documents\American Truck Simulator\mod
  2. if you use -homedir launch parameter then it changes where your mod folder is so please refer to what you set in that parameter

Compatibility

  • Compatible with Game Version 1.56+ and above (your mileage may vary)

Support

This mod pack is provied as it is. I will not provide support for this mod pack if it breaks or damages your game then it is your responsibility. If you're having issue with a particular mod, please contact its mod author to help.
